

## Getting started

This CLI is only for clone my repositories.

```console
  npx L-IGH-T/CreateApp my-app -repo:{name of repository}
```

### All options

|                               Option                              | Description                                                                                                                              |
| :---------------------------------------------------------------: | :-------------------------------------
|  -npm                                                             | Force install with npm. default: yarn
|  -repo:                                                           | Name of repository to clone.
